Get started in 60 seconds

1

Clone the repo

git clone https://github.com/forge-veil/netphantom.git

2

Load in Chrome

Open chrome://extensions, enable Developer mode, click Load unpacked, select the repo folder.

3

Open DevTools

Press F12 on any tab and switch to the NetPhantom panel.

4

Browse normally

The graph builds itself in real time. Click any node to filter. Export with ↓ HAR.
